# RPG Game (Unity)

![Unity](https://img.shields.io/badge/Unity-000000?style=for-the-badge&logo=unity&logoColor=white)
![C#](https://img.shields.io/badge/C%23-239120?style=for-the-badge&logo=csharp&logoColor=white)
![Game Development](https://img.shields.io/badge/Game-Development-green?style=for-the-badge)

An open-world **Role Playing Game (RPG)** developed using **Unity and C#**.

The player explores an interactive environment, fights monsters, and completes quests while progressing through the game world.

---

# 🎮 Gameplay Features

- Player movement and character control
- Environment interaction
- Combat system to fight monsters
- Quest system with objectives
- Open world exploration
- Health and damage system
- Enemy AI behavior

---

# 🛠 Tech Stack

- **Unity Game Engine**
- **C#**
- **Unity Physics**
- **Game Objects and Components**
- **Animations and Scripts**

---

# 🌍 Game Mechanics

### Player
The player can:
- Move around the environment
- Interact with objects and NPCs
- Attack enemies
- Complete quests

### Enemies
Monsters in the game:
- Detect the player
- Attack when the player is nearby
- Take damage and can be defeated

### Quests
The player receives quests such as:
- Defeat monsters
- Explore locations
- Interact with specific objects

Completing quests rewards the player and progresses the game.

# ▶️ How to Run

### 1️⃣ Clone the repository

```
git clone https://github.com/anshdev02/RPG-Game.git
```

### 2️⃣ Open the project in Unity

- Launch **Unity Hub**
- Click **Add Project**
- Select the cloned project folder

### 3️⃣ Open the main scene

Navigate to:

```
Assets → Scenes
```

Open the main scene.

### 4️⃣ Run the game

Press **Play** in the Unity editor.
